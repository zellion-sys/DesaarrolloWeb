Mi Primer JavaScript

Aprendiendo los conceptos básicos de JavaScript

1. Saludo Simple

JavaScript puede mostrar mensajes al usuario.

// Código JavaScript:
function mostrarSaludo() {
  alert('¡Hola! Bienvenido a mi página.');
}

2. Cambiar Color de Fondo

JavaScript puede modificar los estilos de la página.

function cambiarColor() {
  document.body.style.backgroundColor = '#f0f8ff';
}

3. Mostrar y Ocultar

JavaScript puede mostrar u ocultar elementos.

function toggleElemento() {
  var elemento = document.getElementById('elementoOculto');
  if (elemento.style.display === 'none') {
    elemento.style.display = 'block';
  } else {
    elemento.style.display = 'none';
  }
}

¡Este elemento puede aparecer y desaparecer!

JavaScript es mágico ✨

4. Cambiar Texto

JavaScript puede modificar el contenido de la página.

function cambiarTexto() {
  document.getElementById('textoCambiante').innerHTML =
    '¡El texto ha sido cambiado con JavaScript!';
}

Este es el texto original.

5. Calculadora Simple

JavaScript puede realizar operaciones matemáticas.

function sumar() {
  var num1 = parseFloat(document.getElementById('num1').value);
  var num2 = parseFloat(document.getElementById('num2').value);
  var resultado = num1 + num2;
  document.getElementById('resultadoSuma').innerHTML = 'Resultado: ' + resultado;
}
